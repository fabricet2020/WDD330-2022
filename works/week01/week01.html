Learning Local Storage was very interesting. This technique allows users to save data to local storage using multiple keys. The method requires the use of Array when dealing with a long list of values that convert this Array into String and when retrieving it, the code will convert it back from String to Array which invited me to learn JSON as well.

  • LocalStorage uses key-value pairs to store data in the browser (not to the server)
  • Think of an old fashioned phone book as an example
  • The storage limit is 5MB
